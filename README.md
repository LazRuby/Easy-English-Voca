설명 :

엑셀에 원하는 단어를 저장하면 영어 단어와 한글이 번갈아가며 출력됩니다.
이는 자동, 수동 진행으로 설정이 가능합니다.

영어 단어를 보고 한글이 바로 떠오르지 않는다면 모르는 단어입니다.
암기하도록 합니다.

또한, 진행하면서 오답 노트에 기록하면 외우지 못한 단어들만으로 진행이 가능합니다.

계속 반복하여 외워졌다면 단어 리스트에서 지우고 모르는 단어들은 반복하여 봄으로서 영어 단어를 외웁시다.
